Second has a route defined for it in app.routing.ts which will output to the router-outlet in the AppComponent.

It also has its selector tag, app-second in the FirstComponent.

When a user clicks on the First link the browser's location URL changes to match the path segment /start and SecondComponent view will appear with the FirstComponent. This is being delivered through the app-second selector.

However, if a user clicks the Second link, the browser the browser's location URL changes to match the path segment /second and only the SecondComponent will show. This is being delivered through the router-outlet selector.
